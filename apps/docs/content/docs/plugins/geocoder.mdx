# Geocoder Plugin

The Geocoder plugin adds address search capabilities to your map.

## Usage

```tsx
import { createGeocoderPlugin } from '@mapwise/plugins';

await controller.plugins.register(createGeocoderPlugin({
  apiKey: "YOUR_API_KEY", // If required by provider
  provider: "nominatim" // or 'mapbox', etc.
}));
```

The plugin exposes UI components for the search bar, or can be used programmatically to fly to locations.
