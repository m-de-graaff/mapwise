# Draw Plugin

The Draw plugin provides tools to create and edit geometries (Points, LineStrings, Polygons) directly on the map.

## Usage

```tsx
import { createDrawPlugin } from '@mapwise/plugins';

// Register the plugin
await controller.plugins.register(createDrawPlugin());

// Activate a mode
controller.events.emit("plugin:@mapwise/draw:setMode", { mode: "draw_polygon" });
```

## Supported Modes
- `simple_select`: Select and edit existing features.
- `draw_point`: Draw points.
- `draw_line_string`: Draw lines.
- `draw_polygon`: Draw polygons.
