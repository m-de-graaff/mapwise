# Installation

MapWise requires MapLibre GL JS as a peer dependency.

## Package Manager

<Tabs items={["pnpm", "npm", "yarn", "bun"]}>
  <Tab value="pnpm">
    ```bash
    pnpm add @mapwise/core maplibre-gl
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @mapwise/core maplibre-gl
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @mapwise/core maplibre-gl
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @mapwise/core maplibre-gl
    ```
  </Tab>
</Tabs>

## CSS Import

MapLibre GL JS requires its CSS to be imported. Add this to your main CSS file or entry point:

```css
@import "maplibre-gl/dist/maplibre-gl.css";
```

Or import it in your JavaScript/TypeScript entry file:

```typescript
import "maplibre-gl/dist/maplibre-gl.css";
```

## TypeScript Configuration

MapWise is written in TypeScript and provides full type definitions. No additional `@types` packages are needed.

Ensure your `tsconfig.json` has these settings for best compatibility:

```json
{
  "compilerOptions": {
    "strict": true,
    "moduleResolution": "bundler",
    "esModuleInterop": true
  }
}
```

## React Setup

If you're using React, MapWise provides first-class React bindings:

```tsx
import { MapProvider } from "@mapwise/core";

function App() {
  return (
    <MapProvider
      options={{
        style: "https://demotiles.maplibre.org/style.json",
        center: [0, 0],
        zoom: 2,
      }}
      style={{ width: "100%", height: "100vh" }}
    >
      {/* Your map components */}
    </MapProvider>
  );
}
```

### React Version

MapWise supports React 18 and React 19.

## Vanilla JavaScript/TypeScript

MapWise works without React too:

```typescript
import { createMap } from "@mapwise/core";

const container = document.getElementById("map")!;

const controller = createMap(container, {
  style: "https://demotiles.maplibre.org/style.json",
  center: [0, 0],
  zoom: 2,
});

await controller.awaitReady();
console.log("Map is ready!");
```

## Next.js Setup

For Next.js, import the CSS in your root layout:

```tsx title="app/layout.tsx"
import "maplibre-gl/dist/maplibre-gl.css";

export default function RootLayout({ children }) {
  return (
    <html>
      <body>{children}</body>
    </html>
  );
}
```

Since MapLibre uses WebGL and accesses the DOM, ensure you're rendering the map client-side:

```tsx title="app/page.tsx"
"use client";

import { MapProvider } from "@mapwise/core";

export default function MapPage() {
  return (
    <MapProvider
      options={{ style: "https://demotiles.maplibre.org/style.json" }}
      style={{ width: "100%", height: "100vh" }}
    >
      {/* Your components */}
    </MapProvider>
  );
}
```

## Vite Setup

For Vite, import the CSS in your main entry:

```typescript title="src/main.tsx"
import "maplibre-gl/dist/maplibre-gl.css";
import { StrictMode } from "react";
import { createRoot } from "react-dom/client";
import App from "./App";

createRoot(document.getElementById("root")!).render(
  <StrictMode>
    <App />
  </StrictMode>
);
```

## Browser Support

MapWise supports all browsers that MapLibre GL JS supports:

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

WebGL support is required.

## Bundle Size

MapWise is tree-shakeable. If you only use vanilla JavaScript APIs, React bindings won't be included in your bundle.

| Import | Size (gzip) |
|--------|-------------|
| Core only | ~8kb |
| With React bindings | ~10kb |

Note: MapLibre GL JS (~200kb gzip) is a peer dependency and not included in these sizes.
