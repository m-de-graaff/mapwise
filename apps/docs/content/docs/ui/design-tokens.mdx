# Design Tokens

Mapwise UI uses CSS variables for theming, enabling full customization and dark mode support. These tokens map to Tailwind CSS utility classes.

## Colors

### Primary
Used for main actions and active states.

- `--primary`: 222.2 47.4% 11.2%
- `--primary-foreground`: 210 40% 98%

### Secondary
Used for less prominent actions.

- `--secondary`: 210 40% 96.1%
- `--secondary-foreground`: 222.2 47.4% 11.2%

### Destructive
Used for error states and destructive actions.

- `--destructive`: 0 84.2% 60.2%
- `--destructive-foreground`: 210 40% 98%

### Muted
Used for backgrounds and disabled text.

- `--muted`: 210 40% 96.1%
- `--muted-foreground`: 215.4 16.3% 46.9%

## Radius

- `--radius`: 0.5rem (Default border radius for components)
