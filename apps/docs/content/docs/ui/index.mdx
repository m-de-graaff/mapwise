# @mapwise/ui

The UI package provides ready-to-use React components for building map applications. It uses [Tailwind CSS](https://tailwindcss.com/) for styling and [shadcn/ui](https://ui.shadcn.com/) for base components.

## Installation

```bash
npm install @mapwise/ui
# or
pnpm add @mapwise/ui
```

## Core Components

| **[`Map`](/docs/ui/components/map)** | The main map component with markers, popups, and controls. |
| **[`MapShell`](/docs/ui/components/map-shell)** | An application shell layout with sidebars and panels. |
| **[`LayerPanel`](/docs/ui/components/panels)** | A sortable list of layers with controls. |
| **[`MapToolbar`](/docs/ui/components/toolbar)** | A toolbar for map interactions (draw, measure, inspect). |
| **[`AddLayerDialog`](/docs/ui/components/dialogs)** | A dialog to add WMS, WMTS, GeoJSON, and other layers. |
| **[`BasemapSwitcher`](/docs/ui/components/basemap)** | A popover to switch between base maps. |
| **[`WorkspaceMenu`](/docs/ui/components/workspace)** | A dropdown to save/load/export/import map states. |
| **[`Layout`](/docs/ui/components/layout)** | Application shell sub-components (TopBar, StatusBar). |

## Hooks

The package exports [several hooks](/docs/ui/components/hooks) to manage UI state and interactions.

| Hook | Description |
| :--- | :--- |
| **`useLayerList`** | Get formatted layer list for UI. |
| **`useLayerActions`** | Toggle visibility, opacity, reorder, etc. |
| **`useBasemap`** | Manage basemap selection. |
| **`useSelectedFeature`** | Manage feature selection state. |
| **`useMapToolbar`** | Sync toolbar state with map interactions. |
| **`useWorkspaceStorage`** | Save/load workspaces. |

## Usage

Most components are designed to work inside the `MapProvider` context, but the `Map` component itself initializes that context.

```tsx
import { Map, MapShell, LayerPanel } from '@mapwise/ui';

export default function App() {
  return (
    <MapShell
       sidebar={<LayerPanel ... />}
    >
      <Map />
    </MapShell>
  );
}
```
